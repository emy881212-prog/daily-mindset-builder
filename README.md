# Quotes Mindset Builder

A responsive, local-first prototype combining positive quotes, journaling, mood tracking, progress, and a cozy farming mini-game.

Live app: https://9e1f23185eb7abe70f.v2.appdeploy.ai/

## Run locally

```bash
npm install
npm run dev
```

Progress is stored in the browser with `localStorage`. Subscription selection and store purchases use demonstration currency only; no real payments are connected.
